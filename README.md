# Movie Review API

A simple RESTful API built using Node.js core modules to manage movie reviews.  
This project demonstrates CRUD (Create, Read, Update, Delete) operations without using external frameworks.

---

## Features include
- Get all movies
- Get a movie by ID
- Add a new movie
- Update an existing movie
- Delete a movie
- File-based data persistence using JSON

---

## Tech Stack I used

- Node.js (built-in `http`, `fs`, `path` modules)
- JSON for data storage
